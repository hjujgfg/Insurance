Запуск. 
Для работы с гугл картами необходим проект google-play-services_lib, он есть где-то в Android SDK, и ещё его на гитхаб заливаю https://github.com/hjujgfg/google-play-services_lib . чтобы постоянно не менять настройки проекта Insurance (при комитах от разных людей) папка google-play-services_lib должна лежать в той же папке, что и Insurance и сам проект должен быть импортирован в еклипс. 
	Удобнее всего будет, если воркспейс еклипса будет совпадать с папкой, куда гитхаб сохранит Insurance, тогда 
	достаточно 
	1) синхронизировать Insurance
	2) скачать, например, архивом с гитхаба google-play-services_lib и разархивировать его куда-то, только не в воркспейс еклипса
	3) в екслипсе File - Import - Android - Existing android code into workspace -> next
	3a)root directory - указать путь к google-play-services_lib
		ниже в проектах появится соответствующий 
	3b)!!!отметить галкой copy projects into workspace
	3c)Finish
	



Что происходит в программе 
	а) при старте приложения создается файл, куда сохраняются координаты, и запускается сервис, который постоянно считывает координаты.
	б) координаты можно вывести на экран - левый андроид. Очистить файл - средний андроид. 
	в) Запуск просмотра карты - правый андроид. рисуется маршрут. в левом верхнем углу есть кнопка обновить полупрозрачная - перерисовать маршрут(автоматически пока не обновляется)
